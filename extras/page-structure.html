Basic Structure

The page structure includes the header and footer with an underlying grid for the content area.

A corporate home page has a large hero background image and message, with an optional secondary message 126 px tall overlay band at the bottom. This overlay as well as the page footer are anchored to the bottom of the browser window.

Page structure shown with corporate header/footer and home page hero image with overlay band

Note

  • The corporate header and footer are shown. Available header and footer combinations are shown later in this section

Specifications

  • The header and footer background colors extend to the browser window edge, while content is fixed and centered at 975 px wide
  • With the footer anchored to the browser window, deeper page content will scroll while the footer remains stationary and always visible and accessible
  • The optional home page with hero image may also have an overlay band for additional copy with links or thumbnail images of screen captures, videos or book covers
  • If not using the hero image option, the page structure of the home page should follow the same grid as the secondary pages

Content Grid

Grid options for non-corporate pages and any Cengage Learning Marketing websites

The header and footer fill the visible width of the browser window. All other content except for a home page background is limited to the fixed 975 px width. This page grid may also be used for a home page as well as the secondary pages.

Specifications

  • Grid option of 230 px left column for pagelevel navigation, 480 px wide center content column and 230 px wide right column for banners or additional content
  • Grid option of 230 px left column and 724 px wide content (2-column span)
  • Grid option of left 731 px wide content area (2-column span) with 230 px right column for navigation or additional content

Page structure shown with corporate

Left column

Use for secondary or local, page-level navigation.

Main content

The main content may be in the center column, span the left and center columns or the center and right.s

Right column

Use for persistent information like a contact list or a banner ad. A banner ad should expand or support the main page content, not distract from it.

In a layout with content spanning the left and center columns, use the right column for secondary or local, page-level navigation or leave blank.

Specifications

  • Copy can span 2 columns of the grid but should not span all 3 to keep it at a readable maximum line length of 100–120 characters per line, with a preferred count of 55–75 characters per line
  • When content is only in 2 columns with no page-level navigation, use the left and center columns, and leave the right column blank for readable line lengths.
  • Images may run the full 975 px, fit in a single column or span 2 columns
  • If content spans 2 columns, use only left and center or center and right on a single page. Do not mix left spanning and right spanning together on a single page
  • Limit right column banner ads to a maximum of three
  • Right column content should never exceed the depth of the main page content

Header Options

The corporate header is reserved for pages in the corporate site or catalogs. All other websites and pages have three header options.

All headers may use the optional Descriptor text and the optional text area above the tab space, shown below on Option 2.

Specifications

  • Tabbed Header supports a single drop-down menu panel, indicated with the » symbol
  • Tab Bank Header is for navigation without a drop-down menu panel
  • Use the Text Links Header when primary navigation will be in the left content column on the page

Footer Options

The corporate footer includes the country selector. Other websites should use the base footer or a full footer. The footer stays anchored to the bottom of the browser window, regardless of the height of the page content. Even if the page content needs to scroll, the footer always remains visible on screen.

Specifications

  • Corporate footer includes the country selector
  • Full, base and branded footers have the option to include social media links
  • The base footer is the minimum recommended links to be included on all websites and pages
  • Branded footer is used if the logo in the header is anything other than Cengage Learning or an imprint logo. The Cengage Learning logo in the branded footer should link to www.cengage.com
  • Use the white level of the footer for secondary, persistent navigation links specific to a website

Template Files

Design templates

Layered Photoshop files are available for design and mock-up purposes to work within the page grid. Develop and create content graphic assets to fit in the provided framework. The header and footer are built in the HTML template — the only final graphic asset that may be needed for them is when a logo other than the Cengage Learning logo is used.

Development Templates

The variations of the header and footer are built into the available HTML templates and CSS. Select appropriate colors and navigation labels.

Additional resource files:

  • A library of common graphic assets, such as icons
  • JavaScript — to create slideshows, random load of home page hero images, or drop-down menus

Local Navigation

The header should contain the primary navigation for a website. A header option that includes a single drop-down menu is available. Space above the tab bank area may display user information, such as signed in account information. The Descriptor text link should link to the home page for the website.

Specifications

  • Use a header with a single drop-down menu for a navigation list to be accessible from anywhere in the website
  • JavaScript for the drop-down menu is available in our code library (see Quick Links appendix)
  • Other header options are a single tab bank or text links directly on the background color
  • Navigation links on a light background are #0071A2 with an underscore for the hover state

Breadcrumb

A breadcrumb trail offers users a hierarchical way to navigate or trace back the path they have taken in coming to a web page. Use locationbased breadcrumbs, with text links separated by the “greater than” (>) symbol, and in reverse order of the page title. Company Name > Site Name > Section > Page Name/Description.

Example:

Cengage Learning > K-12 > Trades > Construction

Specifications

  • Breadcrumbs fall below the header and above the page headline
  • Breadcrumbs are a user convenience feature and should be used for multi-level sites to provide additional, alternative navigation

Column Navigation

Left column navigation

If primary navigation in the header links to a page that has subsections or subpages, use the left column to house the page-level navigation.

Right column navigation

Use a navigation list in the right column to link to other Cengage Learning and third-party websites or to list downloads related to the page content.

Right column navigation sample

Specifications

  • Left column navigation or right column content should have a 1 px #CCCCCC border, and may include #AF5700 (updated) or #333333 non-linked navigation headers
  • Navigation links are #0071A2 with an underscore for the hover state
  • Indenting navigation lists in right column improves the visual gutter between the center and right columns
