1. The Source

Before linking, be sure to have a code editor (we recommend Sublime Text 2) and some working knowledge of HTML and CSS. We won't walk through the source files here, but they are available to link to and get started.

Connecting to the source

Fastest way to get started: get the compiled versions of our CSS, JS, and images. No docs or original source files.

All of the CSS and JS files are located on the CDN. Simply link to them in your HTML page and get started!

<link href="//cdn.cengage.com/libs/bootstrap/css/cengage-bootstrap.css" rel="stylesheet" media="screen">
<script src="//cdn.cengage.com/js/jquery/jquery-1.7.min.js"></script>

2. File structure

Within the CDN you'll find the following file structure and contents, logically grouping common assets and providing both compiled and minified variations.

If you look at the CDN file structure you'll see something like this:

  boostrap/
  ├── css/
      ├── cengage-bootstrap.css

This is the most basic form of Cengage Bootstrap: compiled files for quick drop-in usage in nearly any web project. We provide compiled and minified all the CSS (cengage-bootstrap.*).

Please note that all JavaScript plugins require jQuery to be included.

3. What's included

Cengage Bootstrap comes equipped with HTML, and CSS for all sorts of things, but they can be summarized with a handful of categories

Docs sections

Scaffolding

Global styles for the body to reset type and background, link styles, grid system, and two simple layouts.

Base CSS

Styles for common HTML elements like typography, code, tables, forms, and buttons. Also includes Font Awesome, a great icon set.

Foundation Components

Basic styles for common interface components like tabs and pills, navbar, alerts, page headers, and more.

4. Basic HTML template

With a brief intro into the contents out of the way, we can focus on putting Bootstrap to use. To do that, we'll utilize a basic HTML template that includes everything we mentioned in the File structure.

Now, here's a look at a typical HTML file:

<!DOCTYPE html>
<html>
  <head>
    <meta http-equiv="X-UA-Compatible" content="IE=Edge">
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Cengage Bootstrap 101 Template</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
    <script src="//cdn.cengage.com/js/jquery/jquery-1.7.min.js"></script>
  </body>
</html>

To make this a Cengage Bootstrapped template, just include the appropriate CSS files:

<!DOCTYPE html>
<html>
  <head>
    <meta http-equiv="X-UA-Compatible" content="IE=Edge">
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Cengage Bootstrap 101 Template</title>
    <!-- Cengage Bootstrap -->
    <link href="//cdn.cengage.com/libs/bootstrap/css/cengage-bootstrap.css" rel="stylesheet" media="screen">
  </head>
  <body class="cengage-boostrap">
    <h1>Hello, world!</h1>
    <script src="//cdn.cengage.com/js/jquery/jquery-1.7.min.js"></script>
  </body>
</html>

IE7 to IE9 Compatibility

If you would like to make compatable with later browsers.

This has the additional <[if IE 8]> and <[if lt IE 9]> statements

<!--[if IE 8]><html lang="en-us" class="ie8"><![endif]-->
<!--[if lt IE 9]><script src="http://html5shim.googlecode.com/svn/trunk/html5.js"></script><![endif]-->

Standard HTML Template

Just copy and paste this template to get started

<!DOCTYPE html>
<html>
  <head>
    <meta http-equiv="X-UA-Compatible" content="IE=Edge">
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Cengage Bootstrap 101 Template</title>
    <!-- Cengage Bootstrap -->
    <link href="//cdn.cengage.com/libs/bootstrap/css/cengage-bootstrap.css" rel="stylesheet" media="screen">
    <!--[if IE 8]><html lang="en-us" class="ie8"><![endif]-->
    <!--[if lt IE 9]><script src="http://html5shim.googlecode.com/svn/trunk/html5.js"></script><![endif]-->
    <link rel="shortcut icon" href="//cdn.cengage.com/libs/bootstrap/img/favicon.png">
  </head>
  <body class="cengage-boostrap">
    <h1>Hello, world!</h1>
    <script src="//cdn.cengage.com/js/jquery/jquery-1.7.min.js"></script>
  </body>
</html>

And you're set! With those files added, you can begin to develop any site or application with Cengage Bootstrap.
